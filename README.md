 # To-Do List App

This is a simple to-do list app built using React Native. It allows users to add, view, and complete tasks.

## Getting Started

To get started, you will need to have Node.js and npm installed on your machine. You can download them from the official website.

Once you have Node.js and npm installed, you can clone the repository and install the dependencies by running the following commands:

```
git clone https://github.com/username/to-do-list-app.git
cd to-do-list-app
npm install
```

After the dependencies have been installed, you can run the app by running the following command:

```
yarn start
```

The app will start running on your local machine. You can open it by going to `localhost:3000` in your browser.
